Portfolio: KnowYour.Fish

Collects data from commercial fishing trips using an Android app, and transforms it with _Ruby on Rails_ into a visual timeline of how a fish was sourced & sold.

  • **Improved database performance** - Trained to audit Rails applications SQL logs for performance opportunities & reduce bottlenecks.
  • **Increased Test Coverage** - Over +1,600 tests existed & ran for over 25 minutes. Reduced run times, measured test coverage increases with _CodeCov_, & integrated _CircleCI & CodeComplete_ automated tests into _Github_.
  • **Front-End HTML/CSS + Javascript.** Expanded existing public & private UI for _Ruby on Rails_ applications using best practices for responsive, mobile-friendly designs & maintaining consistency with existing _Twitter Bootstrap_ layout.
